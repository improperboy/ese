Step-by-Step Guide
Step 1 — Run the setup script
bashcd /home/ubuntu/code
bash jenkins_setup.sh
Wait for it to complete. Your admin password will be saved to jenkins_admin_password.




Step 2 — Complete pipeline-job.xml
Open pipeline-job.xml and add the Checkout stage inside the pipeline script. Find the stages section and add:
groovystage('Checkout') {
    steps {
        git branch: 'main', url: 'git_repo'
    }
}
A typical complete pipeline block looks like:
groovypipeline {
    agent any
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', url: 'YOUR_GIT_REPO_URL'
            }
        }
    }
}
Replace YOUR_GIT_REPO_URL with the actual repo URL from the git_repo variable in your files.





Step 3 — Create/update the Jenkins job from XML
bashJENKINS_URL="http://localhost:8080"
ADMIN_PASS=$(cat /home/ubuntu/code/jenkins_admin_password)

# Create the job using the XML file
java -jar jenkins-cli.jar -s $JENKINS_URL \
  -auth admin:$ADMIN_PASS \
  create-job your-job-name < /home/ubuntu/code/pipeline-job.xml





Step 4 — Build the job and capture output
bash# Trigger the build
java -jar jenkins-cli.jar -s $JENKINS_URL \
  -auth admin:$ADMIN_PASS \
  build your-job-name -s -v

# Wait 10 seconds as instructed
sleep 10

# Get console output and redirect to test1.log
java -jar jenkins-cli.jar -s $JENKINS_URL \
  -auth admin:$ADMIN_PASS \
  console your-job-name > /home/ubuntu/code/test1.log
